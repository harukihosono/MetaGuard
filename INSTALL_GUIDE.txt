================================================================================
               MetaGuard インストールガイド
            セキュリティ警告が出た場合の対処法
================================================================================

■ はじめに
MetaGuard は無料・オープンソースのソフトウェアです。
コード署名証明書を使用していないため、Windows で警告が表示されますが、
安全なソフトウェアです。

================================================================================
1. Windows Defender SmartScreen の警告対処法
================================================================================

【警告画面の例】
┌─────────────────────────────────────┐
│ Windows によって PC が保護されました              │
│                                                  │
│ Windows Defender SmartScreen により、           │
│ 認識されないアプリの起動が阻止されました。       │
│                                                  │
│ [実行しない]  [詳細情報]                        │
└─────────────────────────────────────┘

【対処手順】
1. 「詳細情報」をクリック
2. 「実行」ボタンが表示されるのでクリック
3. インストールが開始されます

================================================================================
2. ファイルのブロック解除方法
================================================================================

ダウンロードしたファイルがブロックされている場合：

【手順】
1. インストーラーファイルを右クリック
2. 「プロパティ」を選択
3. 「全般」タブの最下部にある「許可する」にチェック
4. 「OK」をクリック
5. インストーラーを実行

================================================================================
3. アンチウイルスソフトの警告対処法
================================================================================

【一般的な対処法】
1. 一時的に保護機能を無効化
2. MetaGuard をインストール
3. MetaGuard フォルダを除外リストに追加
4. 保護機能を再有効化

【主要ソフトでの除外設定】
■ Norton
  設定 → アンチウイルス → スキャンとリスク → 除外 → 設定

■ McAfee
  ホーム → PC セキュリティ → ファイアウォール → 除外されたファイル

■ Kaspersky
  設定 → 追加 → 脅威と除外 → 除外 → 追加

■ ESET
  設定 → コンピュータの保護 → 除外

================================================================================
4. 信頼性の確認方法（推奨）
================================================================================

【VirusTotal でのスキャン】
1. https://www.virustotal.com にアクセス
2. インストーラーファイルをアップロード
3. スキャン結果を確認
4. 検出数が少ない（0-2個程度）ことを確認

【ソースコードの確認】
■ GitHub: https://github.com/harukihosono/MetaGuard
■ すべてのソースコードが公開されています
■ Rust 言語で開発（メモリ安全）

================================================================================
5. よくある質問
================================================================================

Q: なぜ警告が出るのですか？
A: コード署名証明書（年間数万円）を使用していないためです。
   ソフトウェア自体は安全です。

Q: 本当に安全ですか？
A: はい。オープンソースで、ネットワーク通信も行いません。
   VirusTotal でのスキャンも推奨します。

Q: 他に似たソフトはありますか？
A: 市販品には同様の機能を持つソフトがありますが、
   MetaGuard は無料で軽量、設定も簡単です。

Q: アンインストールできますか？
A: はい。Windows の「プログラムの追加と削除」から可能です。

Q: 設定ファイルはどこにありますか？
A: インストールフォルダの MetaGuard.ini です。
   メモ帳で編集できます。
