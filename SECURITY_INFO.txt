                    MetaGuard セキュリティ情報
================================================================================

このソフトウェアについて
------------------------
MetaGuard は MT4/MT5 取引プラットフォームの監視・自動起動を行う
オープンソースソフトウェアです。

■ 開発者: Haruki Hosono
■ ライセンス: MIT License
■ ソースコード: https://github.com/harukihosono/MetaGuard
■ 言語: Rust (メモリ安全な言語)

セキュリティ対策について
------------------------
✓ オープンソース - 誰でもソースコードを確認可能
✓ 静的リンク - 外部DLLに依存せず、改ざんリスクを軽減
✓ ウイルススキャン済み - 主要なアンチウイルスエンジンでテスト済み
✓ ネットワーク通信なし - インターネット接続を一切行わない
✓ システム変更最小限 - レジストリへの最小限の書き込みのみ

Windows Defender SmartScreen について
--------------------------------------
このソフトウェアは新しいため、Windows Defender SmartScreen で
「Windows によって PC が保護されました」と表示される場合があります。

これは署名なしソフトウェアに対する一般的な警告であり、
ウイルスやマルウェアを意味するものではありません。

安全に実行する方法
------------------
1. 警告画面で「詳細情報」をクリック
2. 「実行」ボタンをクリック
3. インストールを続行

または

1. ファイルを右クリック
2. 「プロパティ」を選択
3. 「全般」タブの下部にある「許可する」にチェック
4. 「OK」をクリック後、実行

信頼性の確認方法
----------------
■ ソースコード確認
  https://github.com/harukihosono/MetaGuard でソースコードを確認

■ VirusTotal スキャン
  https://www.virustotal.com でファイルをスキャン

■ ハッシュ値確認
  ファイルのSHA256ハッシュ値を公式サイトと照合

動作について
------------
MetaGuard は以下の動作のみを行います：

✓ MT4/MT5 プロセスの監視
✓ 停止時の自動再起動
✓ 設定ファイル（INI形式）の読み書き
✓ Windows 自動起動の設定（オプション）

MetaGuard は以下を行いません：

✗ インターネット通信
✗ 個人情報の収集
✗ システムファイルの変更
✗ 他のプログラムへの干渉
